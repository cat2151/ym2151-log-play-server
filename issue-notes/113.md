# issue issue 103 「Nuked-OPMスレッドへの Windows MMCSS Pro Audio の実装と、 cpal audio_thread_priority フィーチャー有効化、を併用し、オーディオドロップアウトのリスクを低減させる」に実装漏れと実装誤りがあり、codeとtest codeがビルドエラー #113
[issues #113](https://github.com/cat2151/ym2151-log-play-server/issues/113)

# 状況
- Windows local にて、
  - agentに修正させた
    - スムーズだった
      - TDDできるため
- 結果
  - OK

# 原因分析
- GitHub Copilot Coding Agent が、GitHub上の Linux Runner でしか動かないため
  - そこでのagentのTDDは、
    - Linux用feature（Windows用featureなし）となるため
  - MMCSSはWindows用の要素であるため、
    - agentがハルシネーションしたところを、
      - 検知して修正する手段、
        - がなかった
- 対策案
  - GitHub Copilot Coding Agent が、GitHub上の Linux Runner で動く際に、
    - agentがハルシネーションしたとき、
      - それを検知して修正する手段を
        - 調査する
          - 別のissueにて切り分けて実施する

# closeとする
